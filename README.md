# How To Use

|key|control|
|---|-------|
|c|field clear|
|h|move cursor to left|
|j|move cursor to down|
|k|move cursor to up|
|l|move cursor to right|
|o|change cell state|
|p|auto next step(interval 100ms)|
|q|quit|
|s|next step|
|w|save field to save.dump|
