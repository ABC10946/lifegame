This is Conway's game of life that developed in python.

#How To Use

|key|control|
|---|-------|
|c|field clear|
|o|change cell state|
|h|move cursor to left|
|j|move cursor to down|
|k|move cursor to up|
|l|move cursor to right|
|p|auto next step(interval 100ms)|
|s|next step|
|q|quit|
